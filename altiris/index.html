Windows
 
Uninstall through CMD
"C:\Program Files\Altiris\Altiris Agent\AeXAgentUtil.exe" /clean
Open up Task Manager and wait for all processes starting with "AeX" to finish, the proceed to installing the Agent.
 
Install through CMD
 
First open an Explorer windows and browse to "\\bsp-dsksrv-h5.bu.ac.bethel.edu\NSCap\bin\win32\x86", authenticating with your credentials.
Then this command can be run in an Administrator command prompt to install the agent.
"\\bsp-dsksrv-h5.bu.ac.bethel.edu\NSCap\bin\win32\x86\AeXNSCHTTP.exe" -s ns="bsp-dsksrv-h5.bu.ac.bethel.edu" nsweb="https://altiris.its.bethel.edu/altiris" /NOADDREMOVE /s
 
 
Automatic Install URL
http://bsp-dsksrv-h5.bu.ac.bethel.edu/Altiris/NS/Agent/AltirisAgentDownload.aspx, then choose the HTTPs agent installation.  You can use the HTTP on too, but it will change to HTTPS once it checks in.
 
 
 
If all the plugins didn’t install...
  1. Open Run
  2. Navigate to \\bsp-dsksrv-h5.bu.ac.bethel.edu\NSCap\bin\Win64\X64
  3. Enter Bethel credentials
  4. Manually run the installers in the Deployment, Inventory, and Software Management folders
  5. Restart the agent by running “C:\Program Files\Altiris\Altiris Agent\AeXAgentUtil.com” /restart 
 

 

Mac

Restarting the Agent:

Before trying a reinstall, you may want to restart the agent manually by running the terminal command below. 
sudo /opt/altiris/notification/nsagent/bin/aex-helper agent restart

Install or Reinstall the Agent using this dmg file.

Download BU Symantec Management Agent 8 with all plug-ins Installer
 
Uninstalling Manually:
In Terminal, run the following commands. *NOTE: Running the installer using the attached dmg above will uninstall an existing agent first, so this step should not be necessary in most cases.
sudo /opt/altiris/notification/nsagent/bin/aex-uninstall 
sudo rm -rf /opt/altiris

Manual Install (if dmg fails):

  1. Go to the "Go" Menu
  2. Choose "Connect to Server"
  3. Enter "smb://altiris.its.bethel.edu/nscap" and choose "Connect"
  4. Enter your credentials if prompted.
  5. Go to /bin/UNIX/AgentInstall/Mac/x86
  6. Run aex-bootstrap-macosx in a terminal window using sudo.
  7. Follow the prompts
If Using SSH:
  1. mkdir /Volumes/NSCap

  3. /Volumes/NSCap/bin/UNIX/AgentInstall/Mac/x86/aex-bootstrap-macosx
